ExcelToJson 테이블 입력 가이드

엑셀 테이블을 JSON으로 변환하기 위한 작성 규칙과 실행 방법입니다. 처음 작성하는 사람도 이 문서 순서대로 확인하면 됩니다.

읽는 순서
  1. 엑셀 Data 시트 작성 방법
  2. Enum 테이블 입력 방법
  3. 지원 타입과 입력 예시
  4. JSON 변환 실행 방법

1. 엑셀 Data 시트 작성 방법

테이블 엑셀 파일 안에는 Data라는 이름의 시트가 있어야 합니다. 이 Data 시트를 아래 예시처럼 작성합니다.

A B C D E
1 고유 번호 등급 아이콘 이름 값 목록 사용 여부
2 Index Grade Icon value value2
3
4 int GradeType string int[] bool
5 1 LEGENDARY IconItemGem 1, 2, 3 true

작성 설명

테이블 파일은 프로젝트의 Team963\Tables 폴더에 두고, 각 테이블에는 Data 시트가 필요합니다.

행 입력 내용 설명
1행 컬럼 의미 설명 Index, Grade 같은 컬럼 이름이 무엇을 뜻하는지 사용자가 알아볼 수 있도록 적는 칸입니다.
2행 컬럼 이름 JSON의 property 이름이 됩니다. 컬럼 이름이 비어 있는 열은 파싱하지 않습니다.
A2 Index 첫 번째 컬럼 이름은 반드시 Index입니다.
3행 참조 테이블 이름 참조 검사가 필요한 컬럼에만 입력합니다. 필요 없으면 비워둡니다.
4행 데이터 타입 2행에 컬럼 이름을 적었다면 같은 열의 4행 타입도 반드시 입력합니다.
5행부터 실제 데이터 4행 타입에 맞게 값을 입력합니다. Index가 빈 행을 만나면 그 아래는 읽지 않습니다.

위 예시는 JSON으로 변환되면 대략 아래와 같은 형태가 됩니다.

{
  "Index": 1,
  "Grade": 1,
  "Icon": "IconItemGem",
  "value": [1, 2, 3],
  "value2": true
}
Excel 파일이 열려 있어도 변환기는 읽을 수 있습니다. 다만 수정한 내용은 Excel에서 저장한 뒤 변환해야 JSON에 반영됩니다.

2. Enum 테이블 입력 방법

Enum.xlsx는 enum 타입 이름과 값을 정의하는 파일입니다.

A B
1 GradeType ItemType
2 LEGENDARY GOLD
3 RARE GEM

작성 설명

  • 1행에는 enum 타입 이름을 입력합니다.
  • 2행부터 각 enum 타입에 속한 값을 아래로 입력합니다.
  • Data 시트 4행에 GradeType을 입력하면 해당 컬럼은 enum 컬럼이 됩니다.
  • 실제 데이터에는 LEGENDARY, RARE 같은 enum 값을 입력합니다.
  • JSON에는 enum 값의 순서가 1부터 시작하는 숫자로 저장됩니다. 예를 들어 LEGENDARY는 1, RARE는 2입니다.

3. 지원 타입과 입력 예시

4행 타입 엑셀 입력 예시 JSON 출력 예시 주의
int 10 10 숫자 셀로 입력합니다.
float, double 1.5 1.5 소수 입력이 가능합니다.
bool true, false, 1, 0 true Excel의 TRUE/FALSE 또는 숫자 1/0을 사용할 수 있습니다.
string IconGold "IconGold" 빈 칸은 빈 문자열 ""로 출력됩니다.
int[] 1, 2, 3 [1,2,3] 대괄호 없이 쉼표로 구분합니다. 빈 칸은 []로 출력됩니다.
float[], double[] 1.5, 2, 3.25 [1.5,2,3.25] 소수 배열입니다.
string[] apple, banana, sword ["apple","banana","sword"] 쉼표가 값 구분자입니다. 값 자체에 쉼표가 필요한 문자열은 피합니다.
Enum 이름 LEGENDARY 1 Enum.xlsx에 정의된 enum 이름을 타입으로 사용합니다. 값은 enum 순서의 1부터 시작하는 숫자로 저장됩니다.

배열 타입 호환 표기

배열 타입은 아래 표기도 같은 의미로 처리됩니다. 새로 작성할 때는 int[] 형식을 권장합니다.

권장 호환 표기
int[] array[int], int[array], int [ ]
float[] array[float], float[array]
double[] array[double], double[array]
string[] array[string], string[array]

4. JSON 변환 실행 방법

  1. 엑셀 파일을 저장합니다.
  2. 프로젝트의 아래 폴더를 엽니다.
Team963\Tables
  1. ConvertExcelToJson.bat를 더블 클릭합니다.
  2. cmd 창에 변환 로그가 표시됩니다.
  3. 성공하면 아래 폴더에 JSON 파일이 생성됩니다.
Team963\Tables\Json
배치 파일은 성공하거나 실패해도 바로 닫히지 않습니다. 로그를 확인한 뒤 아무 키나 누르면 창이 닫힙니다.

성공 로그 예시

1개의 엑셀 파일을 변환합니다.
[Item_Info] 변환
1개의 JSON 파일을 저장합니다.
Team963\Tables\Json\Item_Info.json
1개 변환/저장 완료
